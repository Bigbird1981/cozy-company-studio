# Cozy Company Studio website

Statische website geschikt voor GitHub Pages, Netlify, Vercel of Cloudflare Pages.

## Structuur

- `index.html` — homepage
- `privacy.html` — privacyverklaring
- `voorwaarden.html` — algemene voorwaarden
- `css/style.css` — styling
- `js/script.js` — interactie
- `images/gallery/` — voorbeeldafbeeldingen
- `images/testimonials/` — testimonial-afbeeldingen
- `documents/` — downloads zoals brochures
- `favicon/` — favicon en manifest

## GitHub Pages

Gebruik relatieve paden zonder leidende slash. Publiceer vanuit de root van de repository.

## Nog invullen vóór livegang

- Definitieve privacyverklaring en voorwaarden juridisch controleren
- Contactformulier koppelen aan backend/formulierdienst
- Eigen domein invullen in `CNAME`, `robots.txt` en `sitemap.xml`
- Eventuele brochure toevoegen aan `documents/brochure.pdf`
